# Chat Segreta (PWA)

Applicazione web di chat privata tra due utenti, protetta da login e cifratura, sempre disponibile grazie a Render.com.

## Funzionalità
- Accesso tramite credenziali
- Interfaccia semplice e responsiva
- Modalità scura attivabile
- Emoji picker integrato
- Timeout per inattività (5 minuti)
- Chat svuotata a ogni disconnessione
- Messaggi criptati con XOR
- PWA installabile su smartphone

## Tecnologie
- HTML, CSS, JS (frontend)
- Socket.io per messaggi in tempo reale
- Express.js per server HTTP
- Manifest.json per supporto PWA

## Come avviare localmente
```bash
npm install
npm start
